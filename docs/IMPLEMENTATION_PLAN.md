# CivicParks Implementation Plan

## v0.1.0

Ship the runtime foundation only:

- FastAPI service and health endpoint.
- Public sample UI.
- Deterministic parks policy Q&A, program/facility answer, registration-link assistance, and maintenance request triage helpers.
- Docs, tests, browser QA evidence, release gate, and package build.

## Later

- Isolated deployment profile if city policy requires it.
- Read-only registration-system adapter design.
- Civic311 handoff adapter design.
- Staff workflow UI.
- Optional local LLM prompt layer through CivicCore.
