# CivicParks Discussion Seeds

## What parks policy sources should ship first?

Which facility rental policies, fee schedules, park rules, league rules, and program catalogs should CivicParks prioritize for staff-reviewed answers?

## What registration-assistance boundaries matter most?

v0.1.0 links residents to existing registration systems but does not process payments or enroll participants. What copy, warning, and review language should appear before staff share registration guidance?

## How should maintenance triage hand off to Civic311?

CivicParks drafts maintenance categories only. What location, urgency, photo, accessibility, and staff-review fields should be included before future Civic311 handoff adapters are designed?
